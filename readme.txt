== Description ==
QnAChat is a powerful live chat customer support plugin for WordPress. It allows you to provide real-time assistance to your website visitors through a chat widget displayed on the front end. Engage with your users, answer their questions, and provide exceptional customer support.

== Features ==
- Live chat widget: Display a chat widget on your website where users can interact with your customer support team.
- Tag-based questions: Users can select tags above the message input field to generate related questions inside the chat for quick answers.
- User messages: Users can send messages directly to the customer support team, and they will receive immediate replies.
- Conversation management: Admin dashboard to manage conversations, including blocking, deleting, adding tags, adding notices, and requesting user info.
- Availability scheduling: Set the availability schedule to define when the "Online" status should be displayed.
- Offline mode: Hide the "Online" status from the chat widget when the customer support team is not available.
- Chat widget customization: Customize the chat widget's theme color, icon, icon size, position, and fine-tune position.
- Auto user info insertion: Automatically update user info with submitted data for a personalized experience.
- Enable for Guests: Allow unregistered users to use the live chat functionality.
- Auto Request Info: Request user information before starting a conversation for better assistance.
- Exclude registered users: Choose to exclude registered users from providing additional info.
- E-mail notifications: Receive e-mail notifications for new conversations to stay updated.
- Disable chat widget: Temporarily hide the chat widget without deactivating the plugin.

== Installation ==
1. Upload the "QnAChat" folder to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Configure the plugin settings through the "QnAChat" menu in the WordPress admin area.
4. Customize the chat widget appearance and functionality according to your preferences.
5. Start engaging with your website visitors and providing live chat support.

== Frequently Asked Questions ==
1. How do I customize the chat widget's appearance?
   You can customize the chat widget's theme color, icon, icon size, position, and fine-tune its position through the plugin settings in the WordPress admin area.

2. Can unregistered users use the live chat functionality?
   Yes, you can enable the live chat functionality for guests, allowing unregistered users to use it.

3. How can I manage conversations?
   You can manage conversations, including blocking, deleting, adding tags, adding notices, and requesting user info, through the admin dashboard provided by the plugin.
